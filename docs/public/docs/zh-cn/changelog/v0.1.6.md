# v0.1.6

## 文档站改进

- 为 Keycast Windows 文档站启用并补充更新日志页面。
- 统一文档 frontmatter 元数据字段，将 `date` 迁移为 `createdAt`。
- 保留 `title`、`description`、`author` 等页面元信息，提升页面头部展示一致性。

## 修复

- 修复 Windows 环境下使用 `CRLF` 换行时，frontmatter 可能被当作正文直接渲染的问题。
- 修复首页和文档页元数据展示不统一的问题。
