# 显示模式详解

Keycast Windows 提供 6 种显示模式，每种模式都针对特定的使用场景优化。

## 模式对比表

| 模式名称 | 说明 | 配置值 | 适用场景 |
|---------|------|--------|---------|
| 仅组合键 | 只显示包含修饰键的组合键 | `combo-only` | 快捷键演示 |
| 仅单个按键 | 只显示单个字母、数字、符号键 | `single-only` | 打字演示 |
| 组合键优先 | 优先显示组合键，无组合时显示单个按键 | `combo-precedence` | 通用场景 |
| 所有按键连续 | 显示所有按键，连续按键时滚动显示 | `all-sequential` | 快速操作 |
| 所有按键队列 | 显示所有按键，持久化直到超时 | `all-sequential-persistent` | 长时间记录 |
| 智能混合 | 根据按键节奏自动切换模式 | `smart-mixed` | 自适应场景 |

## 配置参数说明

| 参数 | 说明 | 默认值 | 取值范围 |
|------|------|--------|---------|
| 组合窗口 | 控制组合键合并显示的时间窗口 | 500 ms | 300-800 ms |
| 最多显示按键数 | 连续按键时最多显示的按键数量 | 5 | 3-20 |
| 连续按键窗口 | 智能混合模式下判定为连续按键的时间窗口 | 400 ms | 200-800 ms |
| 队列空闲超时 | 在**所有按键队列**模式下清空按键的超时时间 | 2000 ms | 200-5000 ms |
| 滚动方向 | 连续按键超出最大数量时的滚动效果 | 淡入淡出 | 横向/纵向/淡入淡出 |
