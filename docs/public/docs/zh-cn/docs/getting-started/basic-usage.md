# 基础使用

## 启动按键监听

> keycast 按键捷监听默认是自动开启的

- 点击主窗口设置界面的 按键监听 的开关按钮
- 使用全局快捷键（默认为Ctrl+Shift+K）


## 查看按键显示

启用按键监听后，尝试以下操作：

- 按 `Ctrl+C` - 显示组合键
- 按 `A` `S` `D` - 根据显示模式显示
- 按 `Ctrl+Shift+Esc` - 显示复杂组合键

屏显窗口会实时显示按键信息。

## 停止按键监听

- 点击主窗口设置界面的 按键监听 的开关按钮
- 使用全局快捷键（默认为Ctrl+Shift+K）


## 退出应用

关闭主窗口：点击主窗口的关闭按钮（×），应用会最小化到托盘。

如果需要完全退出此软件,右键点击托盘图标，选择 **"退出"**

下一步: 查看 [配置入门](/docs/getting-started/configuration) 了解如何自定义按键显示。
