# 下载安装

## 系统要求

<Alert type="warning">
操作系统: Windows 10 (版本 1809) 或 Windows 11 (64 位)

内存: 至少 4 GB RAM

磁盘空间: 约 50 MB 可用空间

</Alert>

## 下载方式

### 官方渠道

推荐从 GitHub Releases 页面下载最新版本：


访问 [Releases 页面](https://github.com/shenjianZ//keycast-windows/releases) 下载：
- `keycast-windows-setup-x.x.x.exe` - 安装程序（推荐）

双击运行 `keycast-windows-setup-x.x.x.exe`，按照向导完成安装

<Callout variant="primary">
提示: 安装程序会自动配置 Windows 防火墙规则，允许应用监听键盘输入。
</Callout>


## 验证安装

启动应用后，系统托盘出现 Keycast 图标

下一步: 查看 [基础使用](/docs/getting-started/basic-usage) 了解如何开始使用。
