# 主题配置

Keycast Windows 提供 12 种精心设计的预置主题，涵盖不同风格和场景。

## 主题分类

### 键盘帽风格

模拟真实键盘按键的视觉效果。

**keycaps-dark** - 深色键盘帽风格，适合深色背景录制

**keycaps-light** - 浅色键盘帽风格，适合浅色背景录制

### 极简风格

简洁纯粹的视觉设计。

**text-only** - 纯文本显示，无背景，最简洁的样式（推荐）

**minimal-light** - 极简浅色主题，清爽干净

### 广播风格

适合直播和录屏场景的高对比度主题。

**broadcast-orange** - 橙色广播风格，高对比度，醒目突出（推荐）

**broadcast-green** - 绿色广播风格，护眼舒适

### 玻璃质感

现代玻璃拟态设计风格。

**glass-soft** - 柔和玻璃效果，现代感十足

**fresh-mint** - 清新薄荷风格，舒适自然

**sky-card** - 天空卡片风格，轻盈通透

### 特殊风格

独特的视觉风格主题。

**terminal** - 终端风格，适合技术演示

**neon-cyan** - 霓虹青色，科技感强烈

**paper-card** - 纸质卡片风格，温馨自然


除了预置主题，你还可以自定义颜色。

## 文字、线条颜色
### 文字颜色

点击 **文字颜色** 色块：
- 支持选择预定定义颜色
- 支持取色器选择
- 支持 HEX 颜色代码（如 `#FF5733`）

### 线条颜色

启用 **显示线条** 后，可以自定义强调线条颜色：
- 用于突出显示按键组合
- 推荐使用与文字颜色对比度高的颜色
