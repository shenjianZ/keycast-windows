# 快捷键设置

使用全局快捷键可以快速切换按键监听状态，无需打开主窗口。

按键监听的全局快捷键默认为`Ctrl+Shift+K`

### 自定义按键监听快捷键

<Alert type="info">
快捷键必须包含至少一个修饰键（Ctrl、Alt、Shift 或 Win）。
</Alert>
在设置页面：

1. 启用 **启用全局快捷键**
2. 点击 **开始录制** 按钮
3. 按下你想要的组合键,松开录制成功


## 快捷键功能

按下按键监听的快捷键时：
- 如果按键监听未启用 → 启用监听
- 如果按键监听已启用 → 停止监听

屏幕会显示通知提示当前状态。

## 禁用快捷键

如果不需要按键监听的全局快捷键功能：

1. 在设置页面找到 **全局快捷键**
2. 取消勾选 **启用全局快捷键"**

## 重置为默认

点击 **默认** 按钮可以将快捷键重置为 `Ctrl+Shift+K`。
