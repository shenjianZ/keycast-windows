# 系统设置

配置应用的行为和系统集成选项。

## 开机自启

### 功能说明

启用后，Keycast Windows 会在系统登录时自动启动。

### 如何设置

在设置页面的 **"启动"** 部分：

1. 勾选 **"开机自启"** 选项
2. 点击 **"保存"** 应用

<Callout variant="primary">
**提示**: 开机自启后，应用会在后台运行，可通过系统托盘图标访问。
</Callout>

### 托盘集成

开机自启与托盘模式配合使用：
- 应用启动后最小化到系统托盘
- 不占用任务栏空间
- 通过托盘图标控制应用

## 语言设置

### 支持的语言

- **简体中文** (zh-CN)
- **English** (en-US)
- **跟随系统** - 自动根据系统语言切换

### 如何切换

在设置页面的 **"语言"** 部分：

1. 点击 **"语言"** 下拉菜单
2. 选择你想要的语言
3. 点击 **"保存"** 应用

<Alert type="success">
语言切换立即生效，无需重启应用。
</Alert>

## 界面主题

### 主题选项

- **浅色** - 明亮的应用界面
- **深色** - 护眼的应用界面
- **跟随系统** - 自动适应系统主题

### 如何切换

在设置页面的 **"样式"** 部分：

1. 点击 **"界面主题"** 下拉菜单
2. 选择你想要的主题
3. 点击 **"保存"** 应用

## 启动时监听

### 功能说明

启用后，应用启动时会自动开始监听键盘输入，无需手动点击"开始"按钮。

### 如何设置

在设置页面的 **"启动"** 部分：

1. 勾选 **"启动时监听"** 选项
2. 点击 **"保存"** 应用

<Callout variant="secondary">
**注意**: 如果启用了开机自启和启动时监听，系统登录后会自动开始按键监听。
</Callout>

## 设置存储

所有设置都会自动保存到：
```
~/.keycast-windows/app-settings.json
```
