# 位置调整

调整屏显窗口在屏幕上的位置，使其适应你的工作流程。

## 位置配置

在设置页面的 **"位置与节奏"** 部分，你可以调整：

### X 坐标

- **范围**: 0 到屏幕宽度
- **说明**: 屏显窗口的水平位置，0表示屏幕左侧边缘

### Y 坐标

- **范围**: 0 到屏幕高度
- **说明**: 屏显窗口的垂直位置,0表示屏幕顶部边缘

## 常见位置设置


- 录屏场景

    推荐将屏显放在屏幕顶部中央，避免遮挡重要内容。
    
    **推荐设置**: X=700, Y=24

- 直播场景

    可以放在屏幕角落，不干扰游戏或应用界面。
    
    **推荐设置**: X=24, Y=24

- 演示场景

    放在屏幕顶部中央，清晰可见。
    
    **推荐设置**: X=700, Y=24


## 实时预览

启用按键监听后，调整位置滑块可以实时预览效果，方便你找到最合适的位置。
