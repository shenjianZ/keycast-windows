# 功能概览

Keycast Windows 是一个功能完善的按键屏显工具，以下是核心功能概览。

## 核心功能

### 1. 按键监听系统

全局键盘输入监听，捕获所有按键事件并实时显示。

### 2. 显示模式系统

提供 6 种显示模式，适应不同的使用场景：

- **仅组合键** - 只显示 `Ctrl`、`Alt`、`Shift` 、`Win`等修饰键组合
- **仅单个按键** - 只显示单个字母、数字、符号键
- **组合键优先** - 优先显示组合键，无组合时显示单个按键
- **所有按键连续** - 显示所有按键，连续按键时水平或纵向滚动
- **所有按键队列** - 显示所有按键，持久化显示直到超时
- **智能混合** - 根据按键节奏智能切换显示模式

详见：[显示模式详解](/docs/advanced/display-modes)

### 3. 主题系统

12 种精心设计的预置主题，涵盖不同风格：

**键盘帽风格**:
- `keycaps-dark` - 深色键盘帽风格
- `keycaps-light` - 浅色键盘帽风格

**极简风格**:
- `text-only` - 纯文本，无背景
- `minimal-light` - 极简浅色主题

**广播风格**:
- `broadcast-orange` - 橙色广播风格
- `broadcast-green` - 绿色广播风格

**玻璃质感**:
- `glass-soft` - 柔和玻璃效果
- `fresh-mint` - 清新薄荷风格
- `sky-card` - 天空卡片风格

**特殊风格**:
- `terminal` - 终端风格
- `neon-cyan` - 霓虹青色
- `paper-card` - 纸质卡片

详见：[主题配置](/docs/user-guide/themes)

### 4. 自定义样式

- **文字颜色** - 自定义文本颜色（RGB/HEX）
- **线条颜色** - 自定义强调线条颜色
- **显示线条** - 控制是否显示强调线条

### 5. 位置与时间配置

- **X/Y 位置** - 精确调整屏显窗口位置
- **组合窗口** - 控制组合键合并的时间窗口
- **连续按键窗口** - 智能混合模式下的连续判定时间
- **队列空闲超时** - 队列模式下的清空超时时间

### 6. 滚动与计数

- **滚动方向** - 横向、纵向或淡入淡出
- **最多显示按键数** - 控制连续按键时的显示数量
- **显示按键计数** - 在连续显示时显示按键数量

## 系统功能

### 1. 全局快捷键

- 自定义系统级快捷键
- 快速切换按键监听状态
- 默认快捷键：`Ctrl+Shift+K`

详见：[快捷键设置](/docs/user-guide/shortcuts)

### 2. 托盘集成

- 最小化到系统托盘
- 托盘图标右键菜单
- 托盘图标状态指示（活跃/待机）

### 3. 开机自启

- 支持开机自动启动
- 与托盘模式配合，实现后台运行

### 4. 国际化

- 内置中英文界面
- 支持跟随系统语言
- 实时切换无需重启

### 5. 自动更新

- 启动时静默检查更新
- 应用内下载更新包
- 一键安装更新

详见：[更新管理](/docs/user-guide/updates)
