# 更新管理

Keycast Windows 内置自动更新系统，让你轻松获取最新功能和修复。

## 自动更新

### 检查更新

应用启动时会自动检查更新：
- 后台静默检查
- 不干扰正常使用
- 发现新版本时显示通知

### 更新流程

1. **检查更新** - 启动时自动检查
2. **下载更新** - 显示下载进度
3. **安装更新** - 一键安装更新包
4. **重启应用** - 完成更新

<Alert type="info">
更新包下载完成后，你可以选择合适的时间点击"安装更新"按钮。
</Alert>

## 更新状态

### 各种状态说明

- **正在检查...** - 正在连接更新服务器
- **发现新版本** - 有可用更新，显示版本号
- **正在下载...** - 正在下载更新包，显示进度
- **已下载** - 更新包已准备就绪
- **当前已是最新** - 没有可用更新
- **检查失败** - 网络连接问题或服务器错误
- **下载失败** - 网络中断或存储空间不足

## 手动检查

如果想在启动时手动检查更新：

1. 打开主窗口
2. 点击 **设置** 按钮
3. 在 **应用更新** 部分点击 **检查更新** 按钮

## 更新配置

### 自动检查

默认启用启动时自动检查。

### 关闭自动检查

如果不想自动检查更新，可以在应用设置中关闭。

## 版本历史

查看更新历史：
1. 访问 [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases)
2. 查看每个版本的更新内容
