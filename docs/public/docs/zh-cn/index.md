# Keycast Windows

<Callout variant="primary">
一个简洁、高效的 Windows 桌面按键屏显工具，为录屏、直播、演示和教学提供清晰的按键反馈。
</Callout>


- 轻量透明屏显窗口，不影响桌面操作，完美适配录屏和直播场景。

- 12 种预置主题，支持自定义颜色和线条配置，满足不同视觉需求。

- 6 种显示模式，从简单的组合键到智能混合模式，适应各种使用场景。

- 位置、时间窗口、滚动方向等全方位配置，打造个性化按键显示体验。

- 内置中英文界面，支持实时切换，国际化用户友好。

- 内置自动更新系统，及时获取最新功能和修复。


## 快速开始

<Alert type="info">
系统要求: Windows 10/11 (64 位)
</Alert>

### 安装

从 [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases) 下载最新的安装包并运行。

### 基础使用

1. 启动应用后，点击 **"开始"** 按钮启用按键监听
2. 按下任意键盘组合，屏显窗口将实时显示
3. 在设置页面调整主题、位置和显示模式
4. 设置全局快捷键，快速切换按键监听状态
