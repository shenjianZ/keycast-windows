# Basic Usage

## Start Keystroke Monitoring

> Keystroke monitoring is enabled by default in keycast

- Click the keystroke monitoring toggle button in the main window settings interface
- Use the global hotkey (default: Ctrl+Shift+K)


## View Keystroke Display

After enabling keystroke monitoring, try the following operations:

- Press `Ctrl+C` - Display combo keys
- Press `A` `S` `D` - Display according to the display mode
- Press `Ctrl+Shift+Esc` - Display complex combo keys

The overlay window will display keystroke information in real-time.

## Stop Keystroke Monitoring

- Click the keystroke monitoring toggle button in the main window settings interface
- Use the global hotkey (default: Ctrl+Shift+K)


## Exit the Application

Close the main window: Click the close button (×) on the main window, and the application will minimize to the tray.

To completely exit the software, right-click the tray icon and select **"Exit"**
