# Installation

## System Requirements

<Alert type="warning">
Operating System: Windows 10 (version 1809) or Windows 11 (64-bit)

Memory: At least 4 GB RAM

Disk Space: Approximately 50 MB of available space

</Alert>

## Download Methods

### Official Channels

It is recommended to download the latest version from the GitHub Releases page:


Visit the [Releases page](https://github.com/shenjianZ//keycast-windows/releases) to download:
- `keycast-windows-setup-x.x.x.exe` - Installer (recommended)

Double-click to run `keycast-windows-setup-x.x.x.exe` and follow the wizard to complete the installation

<Callout variant="primary">
Note: The installer will automatically configure Windows Firewall rules to allow the application to listen for keyboard input.
</Callout>


## Verify Installation

After launching the application, the Keycast icon will appear in the system tray

Next Step: See [Basic Usage](/docs/getting-started/basic-usage) to learn how to get started.
