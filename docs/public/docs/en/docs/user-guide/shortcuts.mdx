# Shortcut Settings

Use global hotkeys to quickly toggle keystroke monitoring status without opening the main window.

The default global hotkey for keystroke monitoring is `Ctrl+Shift+K`

### Customize Keystroke Monitoring Hotkey

<Alert type="info">
Hotkeys must contain at least one modifier key (Ctrl, Alt, Shift, or Win).
</Alert>
On the settings page:

1. Enable **Enable Global Hotkey**
2. Click the **Start Recording** button
3. Press your desired key combination, release to complete recording


## Hotkey Function

When the keystroke monitoring hotkey is pressed:
- If keystroke monitoring is disabled → Enable monitoring
- If keystroke monitoring is enabled → Stop monitoring

A notification will appear on screen indicating the current status.

## Disable Hotkey

If you don't need the global hotkey function for keystroke monitoring:

1. Find **Global Hotkey** on the settings page
2. Uncheck **Enable Global Hotkey"**

## Reset to Default

Click the **Default** button to reset the hotkey to `Ctrl+Shift+K`.
