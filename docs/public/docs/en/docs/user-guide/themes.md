# Theme Configuration

Keycast Windows provides 12 carefully designed preset themes covering different styles and scenarios.

## Theme Categories

### Keyboard Cap Style

Simulates the visual effect of real keyboard keys.

**keycaps-dark** - Dark keyboard cap style, suitable for recording with dark backgrounds

**keycaps-light** - Light keyboard cap style, suitable for recording with light backgrounds

### Minimalist Style

Clean and pure visual design.

**text-only** - Pure text display, no background, the cleanest style (recommended)

**minimal-light** - Minimalist light theme, fresh and clean

### Broadcast Style

High contrast themes suitable for live streaming and screen recording scenarios.

**broadcast-orange** - Orange broadcast style, high contrast, eye-catching (recommended)

**broadcast-green** - Green broadcast style, eye-friendly and comfortable

### Glass Texture

Modern glassmorphism design style.

**glass-soft** - Soft glass effect, full of modernity

**fresh-mint** - Fresh mint style, comfortable and natural

**sky-card** - Sky card style, light and transparent

### Special Style

Unique visual style themes.

**terminal** - Terminal style, suitable for technical demos

**neon-cyan** - Neon cyan, strong technological feel

**paper-card** - Paper card style, warm and natural


In addition to preset themes, you can also customize colors.

## Text and Line Colors
### Text Color

Click the **Text Color** color block:
- Supports selecting predefined colors
- Supports color picker selection
- Supports HEX color codes (e.g., `#FF5733`)

### Line Color

After enabling **Show Lines**, you can customize accent line colors:
- Used to highlight key combinations
- Recommended to use colors with high contrast to text color
