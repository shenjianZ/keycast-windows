# Overview

Keycast Windows is a fully-featured keystroke overlay tool. Here is an overview of the core features.

## Core Features

### 1. Keystroke Monitoring System

Global keyboard input monitoring that captures all key events and displays them in real-time.

### 2. Display Mode System

Provides 6 display modes to adapt to different usage scenarios:

- **Combo Keys Only** - Only displays modifier key combinations like `Ctrl`, `Alt`, `Shift`, `Win`
- **Single Keys Only** - Only displays individual letters, numbers, and symbol keys
- **Combo Keys First** - Prioritizes combo keys, displays single keys when no combo is present
- **All Keys Continuous** - Displays all keys, scrolls horizontally or vertically when keys are pressed continuously
- **All Keys Queued** - Displays all keys, persists until timeout
- **Smart Hybrid** - Intelligently switches display modes based on typing rhythm

See: [Display Modes Explained](/docs/advanced/display-modes)

### 3. Theme System

12 carefully designed preset themes covering different styles:

**Keyboard Cap Style**:
- `keycaps-dark` - Dark keyboard cap style
- `keycaps-light` - Light keyboard cap style

**Minimalist Style**:
- `text-only` - Pure text, no background
- `minimal-light` - Minimalist light theme

**Broadcast Style**:
- `broadcast-orange` - Orange broadcast style
- `broadcast-green` - Green broadcast style

**Glass Texture**:
- `glass-soft` - Soft glass effect
- `fresh-mint` - Fresh mint style
- `sky-card` - Sky card style

**Special Style**:
- `terminal` - Terminal style
- `neon-cyan` - Neon cyan
- `paper-card` - Paper card

See: [Theme Configuration](/docs/user-guide/themes)

### 4. Custom Styles

- **Text Color** - Customize text color (RGB/HEX)
- **Line Color** - Customize accent line color
- **Show Lines** - Control whether to display accent lines

### 5. Position and Time Configuration

- **X/Y Position** - Precisely adjust overlay window position
- **Combo Window** - Time window for merging combo keys
- **Continuous Key Window** - Continuous determination time in smart hybrid mode
- **Queue Idle Timeout** - Clear timeout in queue mode

### 6. Scrolling and Counting

- **Scroll Direction** - Horizontal, vertical, or fade in/out
- **Max Keys to Display** - Control display count during continuous key presses
- **Show Key Count** - Display key count during continuous display

## System Features

### 1. Global Hotkeys

- Customize system-level hotkeys
- Quickly toggle keystroke monitoring status
- Default hotkey: `Ctrl+Shift+K`

See: [Shortcut Settings](/docs/user-guide/shortcuts)

### 2. Tray Integration

- Minimize to system tray
- Tray icon right-click menu
- Tray icon status indication (active/standby)

### 3. Auto-start on Boot

- Supports automatic startup on boot
- Works with tray mode for background operation

### 4. Internationalization

- Built-in Chinese and English interfaces
- Supports following system language
- Real-time switching without restart

### 5. Auto-update

- Silent update check on startup
- In-app download of update packages
- One-click update installation

See: [Update Management](/docs/user-guide/updates)
