# Position Settings

Adjust the position of the overlay window on the screen to fit your workflow.

## Position Configuration

In the **"Position & Rhythm"** section of the settings page, you can adjust:

### X Coordinate

- **Range**: 0 to screen width
- **Description**: Horizontal position of the overlay window, 0 represents the left edge of the screen

### Y Coordinate

- **Range**: 0 to screen height
- **Description**: Vertical position of the overlay window, 0 represents the top edge of the screen

## Common Position Settings


- Screen Recording Scenario

    It is recommended to place the overlay at the top center of the screen to avoid blocking important content.

    **Recommended Settings**: X=700, Y=24

- Live Streaming Scenario

    Can be placed in the corner of the screen without interfering with the game or application interface.

    **Recommended Settings**: X=24, Y=24

- Presentation Scenario

    Place at the top center of the screen for clear visibility.

    **Recommended Settings**: X=700, Y=24


## Real-time Preview

After enabling keystroke monitoring, adjusting the position sliders allows you to preview the effect in real-time, making it easy to find the most suitable position.
