# v0.1.6

## Docs Improvements

- Added release notes content for the Keycast Windows docs site.
- Standardized document frontmatter metadata by migrating `date` to `createdAt`.
- Kept `title`, `description`, and `author` metadata aligned for consistent page header rendering.

## Fixes

- Fixed frontmatter being rendered as page content when documents use Windows `CRLF` line endings.
- Fixed inconsistent metadata presentation across the home page and documentation pages.
