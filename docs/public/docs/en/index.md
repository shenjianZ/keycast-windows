# Keycast Windows

<Callout variant="primary">
A concise and efficient desktop keystroke overlay tool for Windows, providing clear key feedback for screen recording, live streaming, presentations, and teaching.
</Callout>


- Lightweight transparent overlay window that doesn't interfere with desktop operations, perfectly adapted for screen recording and live streaming scenarios.

- 12 preset themes, support for custom colors and line configurations to meet different visual needs.

- 6 display modes, from simple combo keys to intelligent hybrid mode, adapting to various usage scenarios.

- Comprehensive configuration options including position, time window, scroll direction, and more for a personalized key display experience.

- Built-in Chinese and English interfaces with real-time language switching, friendly for international users.

- Built-in automatic update system to get the latest features and fixes in a timely manner.


## Quick Start

<Alert type="info">
System Requirements: Windows 10/11 (64-bit)
</Alert>

### Installation

Download the latest installer from [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases) and run it.

### Basic Usage

1. After launching the application, click the **"Start"** button to enable keystroke monitoring
2. Press any keyboard combination and the overlay window will display it in real-time
3. Adjust themes, position, and display modes in the settings page
4. Set global hotkeys to quickly toggle keystroke monitoring
