# Keycast Windows

<Callout variant="primary">
Built for Windows screen recording, live demos, streaming, and teaching. Keycast Windows stays lightweight, readable, and flexible from the first launch.
</Callout>

- Current stable version: **v0.1.6**
- System support: **Windows 10 / 11 (64-bit)**
- Download: [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases)
- Quick start: [Installation](/docs/getting-started/installation)
- What's new: [View changelog](/changelog)

- Lightweight transparent overlay window that stays out of the way during recording and streaming.

- 12 preset themes, plus custom colors, borders, and line styling for different visual setups.

- 6 display modes, from simple combos to smart hybrid behavior, for different presentation styles.

- Fine-grained control over position, visibility timing, scrolling behavior, and other presentation details.

- Built-in Chinese and English UI with instant switching for bilingual teams and audiences.

- Built-in auto update support helps you stay current with fixes and new capabilities.


## Quick Start

<Alert type="info">
System Requirements: Windows 10/11 (64-bit)
</Alert>

### Installation

Download the latest installer from [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases) and run it.

### Basic Usage

1. After launching the application, click the **"Start"** button to enable keystroke monitoring
2. Press any keyboard combination and the overlay window will display it in real-time
3. Adjust themes, position, and display modes in the settings page
4. Set global hotkeys to quickly toggle keystroke monitoring
