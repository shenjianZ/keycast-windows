# Welcome to the keycast-windows wiki!

**Keycast Windows** 是一个简洁、高效的 Windows 桌面按键屏显工具，为录屏、直播、演示和教学提供清晰的按键反馈。

## 📖 项目概述

Keycast Windows 使用现代化的技术栈构建：
- **前端**: React 19 + TypeScript + Tailwind CSS
- **后端**: Rust + Tauri 2
- **平台**: Windows 10/11 (64位)

## ✨ 核心特性

- **轻量透明** - 不影响桌面操作，完美适配录屏和直播
- **多种主题** - 12 种预置主题，支持自定义颜色
- **智能显示** - 6 种显示模式，适应各种使用场景
- **灵活配置** - 位置、时间窗口、滚动方向等全方位配置
- **双语支持** - 内置中英文界面
- **自动更新** - 内置自动更新系统

## 🚀 快速开始

### 安装

从 [GitHub Releases](https://github.com/shenjianZ/keycast-windows/releases) 下载最新版本并运行安装程序。

### 基础使用

1. 启动应用，点击"开始"按钮
2. 按下任意键盘组合，屏显窗口将实时显示
3. 在设置页面调整主题、位置和显示模式
