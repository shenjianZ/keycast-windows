# 显示模式详解

Keycast Windows 提供 6 种显示模式，每种模式都针对特定的使用场景优化。

## 模式总览

| 模式 | 适用场景 | 组合键 | 单个按键 | 连续显示 |
|------|----------|--------|----------|----------|
| 仅组合键 | 快捷键演示 | ✅ | ❌ | ❌ |
| 仅单个按键 | 打字演示 | ❌ | ✅ | ❌ |
| 组合键优先 | 通用场景 | ✅ | ✅ | ❌ |
| 所有按键连续 | 快速操作 | ✅ | ✅ | ✅ |
| 所有按键队列 | 长时间记录 | ✅ | ✅ | ✅ |
| 智能混合 | 自适应 | ✅ | ✅ | ✅ |

## 各模式详解

### 1. 仅组合键 (combo-only)

**显示行为**: 只显示包含修饰键的组合键

**示例**:
- `Ctrl+C` → ✅ 显示
- `A` → ❌ 不显示
- `Ctrl+Shift+Esc` → ✅ 显示

**适用场景**:
- 软件快捷键演示
- 技术教程录制
- 突出操作重点

**配置建议**:
- 组合窗口: 500 ms

---

### 2. 仅单个按键 (single-only)

**显示行为**: 只显示单个字母、数字、符号键

**示例**:
- `A` → ✅ 显示
- `Ctrl+C` → ❌ 不显示
- `1` `2` `3` → ✅ 显示

**适用场景**:
- 打字教学
- 密码输入演示
- 输入法教学

**配置建议**:
- 无需特殊配置

---

### 3. 组合键优先 (combo-precedence)

**显示行为**: 优先显示组合键，无组合时显示单个按键

**示例**:
- `Ctrl+C` → ✅ 显示
- `A` → ✅ 显示
- `Ctrl+C` `A` `B` → 全部显示

**适用场景**:
- 通用场景
- 日常使用
- 新手推荐

**配置建议**:
- 组合窗口: 500 ms（默认）

---

### 4. 所有按键连续 (all-sequential)

**显示行为**: 显示所有按键，连续按键时滚动显示

**示例**:
- 快速输入 `A` `S` `D` `F` → 水平滚动显示
- 达到最大数量后，旧按键淡出

**适用场景**:
- 游戏操作展示
- 快速打字演示
- 技能展示

**配置建议**:
- 组合窗口: 300 ms
- 最多显示按键数: 5-10
- 滚动方向: 淡入淡出

---

### 5. 所有按键队列 (all-sequential-persistent)

**显示行为**: 显示所有按键，持久化显示直到超时

**示例**:
- 输入 `Ctrl+C` `A` `S` `D` → 持续显示
- 等待"队列空闲超时"时间后清空

**适用场景**:
- 长时间操作记录
- 自动化演示
- 操作回放

**配置建议**:
- 组合窗口: 300 ms
- 最多显示按键数: 10-20
- 队列空闲超时: 2000-5000 ms

---

### 6. 智能混合 (smart-mixed)

**显示行为**: 根据按键节奏自动切换模式

**示例**:
- 慢速输入（间隔 > 连续按键窗口）→ 单独显示
- 快速输入（间隔 < 连续按键窗口）→ 连续显示

**适用场景**:
- 自适应场景
- 高级用户
- 复杂操作

**配置建议**:
- 组合窗口: 500 ms
- 最多显示按键数: 5-10
- 连续按键窗口: 400 ms（关键参数）
- 队列空闲超时: 1000 ms
- 滚动方向: 淡入淡出

## 快速选择指南

| 使用场景 | 推荐模式 |
|----------|----------|
| 软件教程/演示 | 仅组合键 |
| 打字教学/密码输入 | 仅单个按键 |
| 通用场景 | 组合键优先 |
| 游戏/快速操作 | 所有按键连续 |
| 长时间操作记录 | 所有按键队列 |
| 自适应场景 | 智能混合 |

## 配置参数说明

| 参数 | 说明 | 推荐范围 |
|------|------|----------|
| 组合窗口 | 组合键合并时间 | 300-800 ms |
| 最多显示按键数 | 连续显示数量 | 3-20 |
| 连续按键窗口 | 智能模式连续判定 | 200-800 ms |
| 队列空闲超时 | 队列清空时间 | 200-5000 ms |
| 滚动方向 | 滚动效果 | 横向/纵向/淡入淡出 |

## 切换模式

1. 打开设置页面
2. 在"显示模式"下拉菜单中选择
3. 点击"保存"应用更改

更多详情请查看[完整文档](https://shenjianz.github.io/keycast-windows/)。
